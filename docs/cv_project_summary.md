# CV Project Summary

## One-Line Version

Built a two-stage breast cancer multimodal AI system combining mammography screening, pathology foundation-model benchmarking, genomics, clinical features, survival analysis, Isambard GPU workflows, and enterprise healthcare architecture.

## CV Bullet

**Breast Cancer Multimodal AI:** Developed a two-stage medical AI research pipeline using VinDr-Mammo and TCGA-BRCA, with ConvNeXt mammography screening (`AUROC 0.7407`) and CONCH-based pathology + clinical + genomics fusion for PFI risk modelling (`C-index 0.6093 +/- 0.0441`, median split log-rank `p=0.005`); implemented Isambard Slurm workflows, evaluation reports, Streamlit demo, and enterprise guardrail/audit architecture.

Core skills demonstrated:

## Technical Positioning

This project is strongest when positioned as:

> A research-to-enterprise case study showing how multimodal medical AI can be built, evaluated, and wrapped in governance for clinical decision support.

Avoid positioning it as:

> A clinically validated breast cancer diagnostic product.
